# Lotto Recommendation Site Blueprint

## Overview
A modern web application that provides random lotto number recommendations (6 numbers from 1 to 45) with a premium, interactive user experience.

## Features & Design
- **Interactive UI:** Animated lotto balls that appear one by one.
- **Modern Aesthetics:** 
    - Glassmorphism effects and deep multi-layered shadows.
    - Vibrant color palette using `oklch` colors.
    - Responsive layout using CSS Grid and Flexbox.
- **Modern JS:** ES Modules and clean DOM manipulation.

## Current Plan
1. Update `index.html` with the structural layout and Web Component hooks.
2. Update `style.css` with modern CSS features (variables, `:has()`, container queries).
3. Update `main.js` with the lotto generation logic and animation triggers.
